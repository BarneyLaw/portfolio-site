A four-node bare-metal k3s cluster running at home on repurposed Intel NUCs.
The whole thing is provisioned with Ansible from a clean Ubuntu install, so I
can rebuild a node from scratch in minutes.

It serves this site plus a handful of personal services behind Traefik, with
cert-manager handling TLS. Storage is Longhorn for now, though I'm still
deciding whether that's the right call for a cluster this small.
