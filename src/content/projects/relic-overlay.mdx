A real-time market overlay for Warframe that pulls live relic pricing data and
displays it on-screen while you play. It hooks into the Warframe.market API and
a local game-log watcher to figure out which relics you're about to crack.

The overlay runs as a transparent always-on-top Electron window. The log
watcher tails the game's `EE.log`, parses reward screens, and cross-references
each drop against current market orders so I can decide what to keep at a
glance — no alt-tabbing.
